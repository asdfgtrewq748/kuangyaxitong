# MPI算法原理页面计划（MD草案 v0.1）

> 目标：让非算法背景用户在 3~5 分钟内理解 MPI 的逻辑链路、数据来源与可信度，并对热力图输出的“高/低风险区域”建立直觉。

---

## 1. 页面目标与原则

### 1.1 目标
- 明确 MPI 的计算逻辑（输入 → 子指标 → 综合 → 风险等级）
- 提高用户信任（可解释、可追踪、可验证）
- 提升展示品质（图像达到 Science 级别论文配图风格）

### 1.2 原则
- 可解释性优先：每一张图都能回答“为什么”
- 视觉质量优先：图像必须有学术风格的标注、比例与色标
- 交互友好：用户轻点击/滑动即可理解算法变化

---

## 2. 页面结构（信息流）

### 2.1 顶部：一句话总览
- 标题：MPI 矿压影响指标算法原理
- Subtitle：从岩层参数到空间分布的可解释指数

### 2.2 总体流程图（可交互）
- 交互方式：hover 显示每一步数据解释
- 内容：数据输入 → RSI/BRI/ASI → 权重融合 → MPI → 风险等级 → 图像输出

### 2.3 子指标模块（分卡片展示）

#### RSI（顶板稳定性）
- 输入：抗拉强度、关键层数量、岩层结构
- 公式 & 伪代码
- 图：顶板稳定性示意（可做动画：强度变化导致 RSI 变化）

#### BRI（冲击地压风险）
- 输入：采深、硬厚岩层、煤层厚度
- 图：深度增加风险曲线（交互滑块）

#### ASI（支承压力分布）
- 输入：综合刚度、内摩擦角
- 图：应力传递示意 + 剖面图（可动画）

### 2.4 权重解释区
- 默认权重（0.4 / 0.35 / 0.25）
- 可交互：权重调整 → MPI 模拟变化曲线
- “为什么默认权重这样设置”一段解释

### 2.5 风险等级与决策解释
- 低 / 中 / 高风险划分
- 解释“MPI 值 → 施工建议”

### 2.6 示例演算（可信度）
- 选 1~2 真实样例点
- 显示输入参数 → 子指标 → MPI → 风险等级

### 2.7 图像生成逻辑（输出解释）
- MPI 网格如何生成（插值方式）
- 颜色映射与等级划分
- 说明：色带、等值线的含义

---

## 3. 高质量图像与动效要求

### 3.1 图像质量标准（论文级）
- DPI ≥ 300
- 清晰坐标轴 / 标注 / 色标
- 颜色体系统一（ODI 色带）
- 输出格式：SVG + PNG（高分辨率）

### 3.2 动效与交互建议
- **流程图点击**：显示公式与说明
- **滑块**：模拟参数变化 → 曲线 / 热力图变化
- **悬停点**：显示 MPI、RSI/BRI/ASI 细节
- **短动画**：应力传递过程、风险分区变化
