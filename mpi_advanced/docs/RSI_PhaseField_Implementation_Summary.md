# RSI-PhaseField 实现总结

## 概述

成功实现了基于相场断裂模型(Phase-Field Fracture Model)的顶板稳定性指数(RSI)计算模块。

## 核心创新

### 1. 理论创新
- **相场断裂理论**: 使用连续变量φ描述断裂，避免显式追踪裂纹面
- **Francfort-Marigo变分框架**: 基于能量最小化原理预测裂纹扩展
- **Miehe相场模型**: 应变能历史驱动的裂纹演化

### 2. 技术创新
- **断裂力学基础**: 从梁理论升级到断裂力学
- **裂纹预测能力**: 预测裂纹萌生位置和扩展方向
- **损伤量化**: 基于相场分布的定量损伤评估

## 核心方程

### 相场方程
```
G_c * (φ/l_0 + l_0 * |∇φ|²) = 2 * (1-φ) * H
```

其中:
- φ: 相场变量 (0=断裂, 1=完好)
- G_c: 断裂能
- l_0: 相场长度尺度
- H: 历史最大应变能

### 本构关系
```
σ = g(φ) * C : ε

g(φ) = (1-φ)² + η  (退化函数)
```
